This plugin adds a new tab to the category boxes on admin add / edit pages.

== Description ==

This tab has a simple field in which you can enter a keyword and the list is filtered by this keyword.

Simple as the sun, using easy JS.

Especially useful when there are a lot of categories, like in case of a shop.

== Installation ==

To install the plugin, simply install it from the directory, or use the following steps

1. Upload the folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done
